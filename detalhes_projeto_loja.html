Detalhes Técnicos do Projeto - Site da Loja

Linguagens Utilizadas

HTML: Estruturação das páginas do site.

CSS: Estilização e layout das páginas.

Imagens: Utilização de arquivos PNG e JPG para ilustrações e ícones.

Estrutura do Projeto

O projeto é composto de várias páginas HTML que cobrem funcionalidades específicas da loja, como:

  • Página de vendas: Adicionar produtos comprados e processar a compra.
  • Página de controle de estoque: Monitoramento do inventário de produtos.
  • Página de histórico de vendas: Exibição das vendas realizadas, com filtros por data.
  • Página inicial: Página de entrada do site.
  • Página de funcionários: Gerenciamento de informações dos colaboradores.

Estilos e Design

O projeto utiliza CSS para manter um design consistente e atraente. As cores, fontes e layout são definidos no arquivo style.css.

Tempo Estimado de Desenvolvimento

O tempo de desenvolvimento pode variar. Com a estrutura básica pronta demorou em media de 5 a 1 semana semanas.
